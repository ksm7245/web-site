meta 태그

<설명>
(한글이 안될 때)
컴퓨터는 모든 정보를 0과 1로 저장을 하는데
이때 0과 1을 어떻게 저장하는 지에 대한 여러가지 약속들이 있다
약속 중에 하나가 utf-8인데 utf-8이 정확히 뭔진 몰라도 되고 그냥 쓰면 된다?

<예시>
<meta charset = "utf-8">

줄바꿈 태그

<설명>
줄바꿈 태그에는 br과 p가 있는데
br은 Break의 약자로 다음 칸으로 넘어가는 줄바꿈 태그이고
p는 Paragraph의 약자로 문단을 만들어주는 줄바꿈 태그이다
br은 닫을 필요 없이 <br>한 번만 써주면 되는데
p는 <p>로 열고 </p>로 닫아줘야 한다

<예시>
문장 끝부분에 <br>를 붙여줌
<p>
문단
</p>

li 태그

<설명>
li는 list의 약자로 리스트를 만들어주는 태그이다
li의 부모태그로는 ol와 ul이 있는데
ol은 ordered list의 약자로 순서가 정해진 리스트이고
ul은 unordered list의 약자로 순서가 정해지지 않은 리스트이다

<예시>
<ol>
<li>
1.HTML
2.CSS
3.JavaScript
</li>
</ol>

<ul>
<li>
HTML
CSS
JavaScript
</li>
</ul>
